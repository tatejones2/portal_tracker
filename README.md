# Portal Board

A private React + Vite transfer portal school tracking dashboard for college baseball recruiting.

## Run locally

```bash
npm install
npm run dev
```

In a second terminal, run the optional AI research API:

```bash
cp .env.example .env
npm run api
```

The OpenAI key must be stored as `OPENAI_API_KEY` and is only read by the server-side API script.
